# 日本市場セクターローテーション

日本の株式市場（TOPIX-17業種）のデータを解析し、現在の景気循環（セクターローテーション）を可視化するシステムです。

## 仕組み

1.  **自動実行:** 平日の夕方にGitHub Actionsが起動。
2.  **データ取得:** 過去2年分の株価を取得し、200日移動平均線からの乖離率を計算。
3.  **チャート生成:** 景気敏感度が高い「精鋭12業種」の動きを合成し、市場の重心（景気フェーズ）を算出。

---

## 景気サイクルの判定ロジック

市場を時計回りの4つの季節（フェーズ）に分類して判定します。

| フェーズ | 位置 | 担当セクター | 特徴 |
| :--- | :--- | :--- | :--- |
| **回復期** | 左上 (北西) | 電機精密、情報通信、建設資材 | 金利低下・不況脱出・ハイテク始動 |
| **好況期** | 右上 (北東) | 自動車、機械、商社卸売 | 業績拡大・輸出産業・設備投資 |
| **後退期** | 右下 (南東) | 銀行、金融、エネルギー | 金利高・インフレ・逆金融相場 |
| **不況期** | 左下 (南西) | 医薬品、食品、小売 | 逆業績相場・ディフェンシブ優位 |

**※除外セクター:**
ノイズを防ぐため、動きが重複する「鉄鋼・非鉄」「素材・化学」「不動産」「運輸・物流」「電力・ガス」は計算から除外しています。
